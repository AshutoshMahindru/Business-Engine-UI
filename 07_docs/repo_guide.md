# Repo Guide

## Folder meaning
- `01_schema` = source business structure
- `02_mermaid_system` = screen and flow diagrams
- `03_ui_generation` = prompts and generation rules
- `04_design_system` = tokens, components, layouts, patterns
- `05_prototypes` = generated UI outputs and review history
- `06_frontend` = implementation layer
- `07_docs` = narrative and reference
- `08+` = execution system layers

## Golden rule
Every screen should evolve through:
diagram → prompt → prototype → refined prototype → frontend
